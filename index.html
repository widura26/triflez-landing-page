Slide 4: Privacy Framework

Privacy Policy

Last Revision: June 13, 2026
Google Verification Notice: This software specifically uses the https://www.googleapis.com/auth/drive.file scope strictly to persist highly secure backups. This program does not have permissions or technical utilities to query or modify other folders in your Google Drive cloud space.

1. Absolute Privacy Commitment

Triflez guarantees absolute data ownership. The application does not deploy remote analytical modules, capture identity parameters, or forward client token seeds. All operational components occur on your local web browser engine.

2. Safe Storage Protocols

Any seed configuration datasets entered into the user interface are instantly committed to Chrome's local isolated storage area, protected by highly robust AES-256 encryption. Your decryption credentials are never transmitted, persisted, or known by the developers.

3. Drive Storage Integration Parameters

The Drive API is exclusively integrated to provide robust recovery channels:

  • To format and construct an isolated JSON document inside your storage partition containing your encrypted credentials.
  • To parse this container file when restoration is requested by the user.
  • To perform timestamp audits to prevent local configurations from overwriting cloud records.

4. Zero Third-Party Routing

No telemetry variables, database setups, or user-specific records are collected, leased, or distributed. Triflez operates entirely independently of tracking servers or programmatic marketing profiles.

5. Core Inquiries

If you have questions regarding the structural isolation rules, database schemas, or cryptographic methodologies utilized, kindly coordinate with our verification compliance officer at support@triflez.com.
